### 🏭 Smart Manufacturing Solutions

#### OEE 智慧工廠稼動率分析模組
**系統規劃分析 & 後端開發**

一套即時監控生產效率的分析系統，協助傳統製造業掌握產線稼動率（OEE）與生產數據。

- **核心技術**  
  `Microservices` `Node.js (Express)` `MongoDB`

- **關鍵成果**
  - 🎯 設計 OEE 三大指標（可用率、效能率、良率）的即時計算邏輯
  - 🚀 RESTful API 設計
  - 📊 支援多維度數據切分（機台、產線、時段），提升決策效率

---
